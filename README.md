# WS Regulab Solutions - Training Landing Page

A minimalist and professional landing page for regulatory affairs and formulation training services.

## 🚀 Features

- **Modern UI/UX**: Built with React, Vite, and TailwindCSS for a smooth, responsive experience.
- **Dynamic Animations**: Seamless transitions powered by Framer Motion.
- **Vercel Ready**: Optimized for direct deployment to Vercel.

## 🛠️ Tech Stack

- **Framework**: React 19
- **Build Tool**: Vite
- **Styling**: TailwindCSS 4
- **Animations**: Framer Motion

## 💻 Local Development

Follow these steps to run the landing page locally:

1. **Clone the repository**:
   ```bash
   git clone https://github.com/me-yeatz/WS_Landing-Page.git
   cd WS_Landing-Page
   ```

2. **Install dependencies**:
   ```bash
   npm install
   ```

3. **Run the development server**:
   ```bash
   npm run dev
   ```

4. **Build for production**:
   ```bash
   npm run build
   ```

## 🌐 Deployment

This project is optimized for deployment on Vercel. Simply connect your GitHub repository and Vercel will handle the rest.
